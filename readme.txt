Currency Converter using Exchange rate api
Author: Christian Brown 07/19/23
=============

Description
------------
Currency exchange console application. This application request real time exchange rates from "https://api.exchangerate.host/latest"
    The application requests this data as soon as the application is started.

Installation
------------
    List of Dependencies: CURL, nlohmann/json
    Install both packages.

Usage
-----
    1. Open terminal
    2. "cd" into build directory
    3. run "make" then "./test" to run application.

Make sure to cover various use cases and any configuration options.
